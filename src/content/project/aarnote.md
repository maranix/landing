Aarnote is a cross-platform note-taking app featuring a sleek dark theme with glassmorphism effects, built for both iOS and Android.

## Tech Stack
- **Framework**: React Native 0.81.5 with Expo ~54.0
- **Language**: TypeScript 5.9.2
- **State & Storage**: Zustand 5.0.8, react-native-mmkv 4.0.1
- **Animations**: react-native-reanimated 4.1.1

## Core Features
- Local-first instant access with secure MMKV storage.
- Password hashing via SHA-256 for security.
- Rich editing including image support.
- Smooth transitions and 60fps micro-interactions.
